IFACEINTSHARED

IFACEEXTDIRECT

IFACEEXTSHARED

IFACEINTDIRECTMSG
	EN	Direct access to the internal audio interface has been configured. Variable output sample rates are enabled, but only SqueezePlay will have access to the interface while running.\n\nYou will need to relaunch SqueezePlay for this setting to take effect.

IFACEINTSHAREDMSG
	EN	Access to the internal audio interface through the mixer has been configured. The output sample rate is fixed at 44.1 kHz, but all software on this device can access the interface.\n\nYou will need to relaunch SqueezePlay for this setting to take effect.

IFACEEXTDIRECTMSG
	EN	Direct access to an external audio interface has been configured. Variable output sample rates are enabled, but only SqueezePlay will have access to the interface while running.\n\nYou will need to relaunch SqueezePlay for this setting to take effect.

IFACEEXTSHAREDMSG
	EN	Access to an external audio interface through the mixer has been configured. The output sample rate is fixed at 44.1 kHz, but all software on this device can access the interface.\n\nYou will need to relaunch SqueezePlay for this setting to take effect.
